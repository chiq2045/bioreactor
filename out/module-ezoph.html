Module: ezoph

Communicate with the EZO pH device and sensor
Properties:
Name Type Description
i2c object an i2c object
address int address of the device
ph String the last pH value read
timeout int the time taken to carry out operation, in ms. defaults to 900, the longest time needed for any operation.

Returns:
value - the output of the device. this can be anything from the pH to the current status of the device.
